
Тестове завдання на посаду Unity developer
Мета:
Реалізувати фізично коректну модель польоту вертоліта.
Загальний опис задачі:
Потрібно створити невеликий playable prototype, у якому користувач може керувати літальним апаратом у 3D-сцені.
Основний акцент:
фізика руху;
керування;
стабільність польоту;
структура коду;
адекватна архітектура.
Мінімальні вимоги:

1. Основний рух:
зліт;
посадка;
зависання;
рух вперед/назад;
рух вліво/вправо;
поворот по yaw;
нахил pitch/roll.

3. Фізика (Не використовувати готові flight-controller assets.):
підйомна сила;
тяга;
гравітація;
інерція;
drag;
torque.

5. Керування (використати Input System)
WASD — рух;
Space / Ctrl — висота;
QE — поворот.

Технічні вимоги:
Рекомендована версія:
Unity 6000.2.7f2 HDRP.
Срок виконання:
1 тиждень з моменту отримання данного завдання.
Візуал:
Можно реалізувати на примітивах.
